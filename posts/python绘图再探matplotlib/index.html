再探matplotlib与商务图表的绘制

这里还是以前期文章中讲到的六种常用图表为例来说明matplotlib库第二种绘图方式，在各种图表绘制之前，小编先进行一下共用性的代码设置：

# 导入相关的库
import numpy as np
import pandas as pd
import matplotlib as mpl
import matplotlib.pyplot as plt
import pandas_datareader.data as web

# 进行绘图风格以及绘图显示格式的设置
plt.style.use('seaborn') # 使用seaborn图表风格
plt.rcParams['font.sans-serif']=['SimHei'] #用来正常显示中文标签
plt.rcParams['axes.unicode_minus']=False #用来正常显示负号

上面代码的前一部分主要是导入了后文进行分析所需要的第三方库，当然最后一个库的导入并不是必须的，小编这里导入它是为了在后面分析中获取一些股市价量数据，诸位这一步并不是必须的。后部分主要是对图表的显示风格以及显示格式进行了设置，请注意最后两条语句是很有必要的（如果你的图表中包含中文和负号的话）。鉴于在前期的文章中已经对各种图进行了比较详细的介绍，后文的介绍将以代码和图表的形式进行展现，只在绘制第一个图的时候对第二种绘图方式予以介绍，后面如无必要不再过多予以文字说明，下面进行第一类图形折线图的绘制。

折线图的绘制

折线图包括传统的折线图，当然广义上也包含面积图，下面先介绍一般的折线图绘制方法，而后进行面积图绘制方法的介绍。

  • 折线图
x = np.linspace(0,np.pi,1000)
y1 = np.sin(x)
y2 = np.cos(x)
fig = plt.figure()
ax = plt.axes()
ax.plot(x,y1,label='sin(x)')
ax.plot(x,y2,label='cos(x)')
ax.legend()
ax.set(xlabel='Value x',ylabel='$sin(x) & cos(x)$',title='正弦和余弦图绘制')
fig.savefig('正弦余弦图.png')

这些命令中的多数代码前面的文章中已有说明，重点对其中的几条新的命令方式进行说明，即代码中的第4-5行，尤其是第五行代码。第四行代码用于创建了一个图表对象，而第五行代码则用于创造了一个绘图区，也就是通常所了解到的图表编辑区，一般来说，一个图表对象可以包含多个绘图区，至少有一个绘图区的图表对象才有意义。而其它命令则主要是将原来的plt方式换成了ax方式，所以相信这一部分你能很快掌握，同时最后一条命令用于将创建的图表保存在电脑上。下面进行面积图介绍。

  • 面积图
x = np.arange(5)
y1 = np.array([2,3,5,8,13])
y2 = np.array([2,4,6,8,10])
y3 = np.array([1,3,5,7,9])
labels = ['斐波那契数','偶数','奇数']
fig,ax = plt.subplots()
ax.stackplot(x,y1,y2,y3,labels=labels)
ax.legend(loc=2)
ax.set(title='面积图')
fig.savefig('面积图绘制.png')

条形图绘制

条形图分为水平条形图和竖直条形图（又称之为柱状图），根据是否堆积又分为堆积条形图和非堆积条形图，下文小编先绘制非堆积的水平条形图和竖直条形图，而后再给出一个绘制堆积条形图的方法。

  • 非堆积水平条形图
values = np.random.randint(10,30,6)
labels = ['A','B','C','D','E','F']
fig,ax = plt.subplots()
ax.barh(np.arange(6),values)
ax.set_yticks(np.arange(6))
ax.set_yticklabels(labels)
ax.set(xlabel='values',ylabel='labels',title='水平条形图绘制')
fig.savefig('水平条形图.png')

  • 非堆积竖直条形图
values = np.random.randint(10,30,6)
labels = ['A','B','C','D','E','F']
fig,ax = plt.subplots()
ax.bar(np.arange(6),values)
ax.set_xticks(np.arange(6))
ax.set_xticklabels(labels)
ax.set(xlabel='labels',ylabel='values',title='竖直条形图绘制')
fig.savefig('竖直条形图.png')

  • 堆积条形图
values1 = np.random.randint(10,20,4)
values2 = np.random.randint(18,25,4)
values3 = np.random.randint(21,30,4)
labels = ['A','B','C','D']
fig,ax = plt.subplots(2,1)
ax[0].bar(np.arange(4)-0.2,values1,width = 0.2,color='r')
ax[0].bar(np.arange(4),values2,width = 0.2,color='g')
ax[0].bar(np.arange(4)+0.2,values3,width = 0.2,color = 'b')
ax[0].set_xticks(np.arange(4))
ax[0].set_xticklabels(labels)
ax[0].set(xlabel='lables',ylabel = 'values')
ax[1].barh(np.arange(4)-0.2,values1,height=0.2,color='r')
ax[1].barh(np.arange(4),values2,height=0.2,color='g')
ax[1].barh(np.arange(4)+0.2,values3,height=0.2,color = 'b')
ax[1].set_yticks(np.arange(4))
ax[1].set_yticklabels(labels)
ax[1].set(xlabel='values',ylabel = 'labels')
plt.suptitle('堆积条形图的绘制')
fig.savefig('堆积条形图绘制.png')

一个综合示例

最后小编以一个综合的示例结束本期文章，在这个示例中小编使用苹果公司近五个月的股价数据进行相关图表的绘制，其中使用了pandas-datareader库来获取绘图中所需要的相关数据，这里小编就不对其进行过多介绍了。下面见代码与图表：

# 获取苹果公司近五个月股价数据
df = web.DataReader('AAPL','yahoo',start='2018-01-01',end='2018-05-12') # 第一个参数是苹果在交易所代码，第二个参数是数据来源，后两个参数代表获取的开始于结束时间
fig,ax = plt.subplots(2,2,figsize=(16,9))
ax[0,0].plot(df.Open,color='r',label='开盘价')
ax[0,0].plot(df.High,color='g',label='最高价')
ax[0,0].plot(df.Low,color='b',label='最低价')
ax[0,0].plot(df.Close,color='k',label='收盘价')
ax[0,0].legend()
ax[0,0].set(ylabel='价格',title='苹果价格数据')

ax[0,1].bar(df.index,df.Volume)
ax1 = ax[0,1].twinx()
ax1.plot(df.Close,color='g')
ax[0,1].set(ylabel='成交量',title='苹果量价数据')
ax1.set(ylabel='收盘价')

ax[1,0].scatter(df.Volume/100000,df.Close,color='r')
ax[1,0].scatter(df.Volume/100000,df.Close,color='b')
ax[1,0].set(xlabel='成交量（十万）',ylabel='收盘价',title='苹果量价关系')

ax[1,1].boxplot((df.Open,df.High,df.Low,df.Close,df['Adj Close']),labels=('开盘价','最高价','最低价','收盘价','调整价'))
ax[1,1].set(xlabel='五价',ylabel='价格',title='苹果相关价格箱线图')

plt.suptitle('苹果价量关系综合图表')
fig.savefig('综合示例.png')
