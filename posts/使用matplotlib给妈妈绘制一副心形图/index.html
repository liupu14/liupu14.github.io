我们都知道，笛卡尔当年为了向自己的小女友示爱创造了心形图，所以我们何尝不能使用这种心形图向自己的母亲表达爱意呢？心形图的绘制无非就是数学公式的可视化，因此小编这里先介绍一些这些爱的方程式，而后使用python代码实现心形图的绘制。

  • 心形图的极坐标方程
    • 水平方向：$r = a(1-cos(\theta),a>0$
    • 竖直方向：$r = a(1-sin(\theta),a>0$
  • 心形图的直角坐标方程
    • 水平方向：$x^2+y^2+ax=a\sqrt{x^2+y^2}$
    • 竖直方向：$x^2+y^2-ax=a\sqrt{x^2+y^2}$
  • 心形图的参数方程 $ x = a(2cos(t) - cos(2t))$ $ y = a(2sin(t) - sin(2t))$

了解了心形图的方程系统，那么下面就可以开始进行心形图的绘制了，小编这里采用python中matplotlib库来实现心形图的绘制。

# 导入绘图中所需库
import matplotlib.pyplot as plt   
import numpy as np  
import math  
# 定义绘图函数
def drawHeart():  
    t = np.linspace(0, math.pi, 1000)  
    x = np.sin(t)  
    y = np.cos(t) + np.power(x, 2.0/3)  
    plt.plot(x, y, color='red', linewidth=2)  
    plt.plot(-x, y, color='red', linewidth=2)  
    plt.xlabel('t')  
    plt.ylabel('h')  
    plt.ylim(-1, 1.6)  
    plt.xlim(-1.1, 1.1)   
    plt.axis('off')
# 调用函数
>>> drawHeart()

你是一个合格的儿女吗？

上文小编引导大家完成了心形图的绘制，下面小编将谈一下我们目前所处于的实际，我们真的是合格的儿女吗？这里小编不说其他的，仅仅以我们在各个阶段中每年所能陪同父母的时间来进行说明。小编将我们的一生划分成六个阶段：幼童及小学；初中；高中；大学；工作阶段以及退休阶段。每个阶段以年均陪同父母的天数作为时间的度量标准。幼童及小学时期，我们基本是陪同在父母身边的，我们一年中最少能够陪同你父母360天；初中时期，我们很多人可能需要住校，因此一般是每周在家两天，一年有53周外加各种假期，我们能够陪同父母的时间大概是160天；高中时期，基本是每个月能够回家一次（而且寒暑假需要在校进行补课），因此我们在高中时期最多每年能够陪同父母80天；大学时期一般是半年回去一次，因此我们主要是在寒暑假时才会陪同在父母身边，每年最多60天；工作之后，我们基本是每年回去一次，年假一般不会超过十天，考虑到其它假期，我们每年最大陪同父母15天；等我们退休了，你觉得我们还能够陪同父母吗，此时我们能够陪同父母的时间大致是0，见下表总结：

人生阶段 陪同父母时间 时间占比
幼童及小学时期 360 $\sim$100%
初中阶段 160 43.8%
高中阶段 80 21.9%
大学阶段 60 16.4%
工作阶段 15 4.1%
退休阶段 0 0%

可以很明显地看出，随着我们逐步长大，我们陪同父母的时间越来越短，而且我们真正陪在父母身边比较长的阶段主要是在幼童及小学，以及初中时期，这个时段与其说是我们陪同父母，倒不如更加确切地说是父母浪费他们的最宝贵时间陪在了我们身边，一旦当我们心智开始成熟之后，即在高中及之后，我们便开始了越发远离父母的趋势。

古人云：父母在不远游。然而从中学时代开始，我们便开始了离开父母身边的历程，而且距离越来越来，及至现在已有千里之距。经济学家都是一群聪明的人，他们能够研究各种信息不对称的情况，但是他们却从来不会去了解父母与子女之间爱的不对称现象。随着时间的流逝，父母增加的是日益花白的发丝以及对于子女的无限挂念，而我们增加的却是永不满足的野心以及征服一切的欲望，这就使得我们和父母间不论从物理距离还是从心理距离方面都渐行渐远。

所以我想说的是，你现在真的还觉得你是一个合格的子女吗？作为一个合格的子女，你不能常伴父母身边尽孝；即使你不得以常年在外，你又多久才会选择和父母好好通一次电话，随便唠唠日常的一切呢？我们总是把忙当作我们应对一切的借口，把追求事业的成功以及声誉的显赫作为我们懈怠其它的工具，然而我们何时又真的去反思过我们的成长又何尝不是父母逝去他们的宝贵时光换来的呢？所以，每念及此，小编总是会眼噙泪水，无法释怀！对于父母，我亏欠他们太多；作为儿子，我终究不是一个合格的儿子。
