上期文章和大家简单聊了一下怎么使用python去爬取酷狗Top500排行榜的相关数据，了解了python爬取多页面网页的基本原理，也让大家了解到了目前流行的一些歌曲情况，后续文章会进一步介绍python的网络爬虫。但是，本期对于爬虫的介绍将先放一下，因为小编似乎很久没有和大家聊图表的绘制了，小编觉得很有必要讲一下这个。因此本期文章将和大家接着聊一下图表的绘制，主要是散点图的绘制。在本期的介绍中，小编将先介绍最基本的matplotlib实现过程，而后介绍python中绘制静态图表的优秀库seaborn的实现过程，最后将介绍怎么使用R语言来实现散点图的绘制。下面正式开始！

matplotlib与散点图

作为祖师级别的绘图库，matplotlib奠定了其它python绘图库的基础，因此使用python绘制图形自然离不开对于matplotlib库的相关介绍。虽然，相比其它绘图库，matplotlib库绘制图形稍显繁琐，然而熟练地掌握了matplotlib库，才能随心所欲地对各种python图形进行格式设置，所以在小编以后介绍的任意图表绘制中，都将先介绍怎么使用matplotlib库实现，而后再介绍各种更加快速地绘制方法。本期文章主要讲解散点图的绘制，鉴于散点图可能是你遇到的最多的一种图表形式，大家应该已经足够熟悉了散点图的相关概念，所以这里就不再长篇大论散点图了，仅仅介绍一下散点图的绘制过程。如果各位对于小编前期的文章有所了解，那么就会知道使用python绘制散点图存在多种方法，不管使用那种方法，其绘制过程都不是很复杂。这里首先讲一下怎么使用matplotlib绘制最基本的散点图。

# 导入相应的库，并且设置绘图嵌入环境
import matplotlib as mpl
import numpy as np
import matplotlib.pyplot as plt
%matplotlib inline
plt.rcParams['font.sans-serif'] = ['SimHei']  # 中文正常显示
plt.rcParams['axes.unicode_minus']=False  # 负数正常显示

# 创建数据集
x = np.random.randn(100)
y = 2 * x + np.random.randn(100)

# 使用matplotlib绘制散点图
fig = plt.figure()
ax = fig.add_subplot(111)
ax.scatter(x,y)
plt.title('matplotlib与散点图')
plt.xlabel('x')
plt.ylabel('y')

使用matplotlib库绘制散点图最关键的一步就是ax.scatter，其它命令主要是对图形进行格式化或者进行图形的显示设置等，同时要提醒各位的是，使用matplotlib库绘制图形存在两种方式，小编这里混合使用了两种方式，对于两种方式的介绍小编在前期的文章中已经介绍过了，所以这里不再赘述，以后小编会以方便为基准混合使用这两种绘图方式。在对matplotlib库绘制散点图有了基本了解后，下面转入到更加高级的绘图系统。

seaborn与散点图

seaborn是基于matplotlib库开发出的一种更加高级的绘图库，它的开发方便了python对于静态图表的绘制，因此小编建议在绘制静态图表时使用这种绘图系统，而对于图形样式的设置则可以参照matplotlib的语法。下文以seaborn库自带的iris数据集为例来进行散点图绘制的讲解，关于这个数据集相信看过小编文章的朋友已经有所了解了，这里就不再赘述。下面首先绘制一个最基本的散点图。

import matplotlib.pyplot as plt
import seaborn as sns
from seaborn import load_dataset
iris = load_dataset('iris')
plt.rcParams['font.sans-serif'] = ['SimHei']  # 中文正常显示
plt.rcParams['axes.unicode_minus'] = False  # 负数正常显示
sns.set(style = 'whitegrid')
%matplotlib inline
sns.relplot(x = 'sepal_length',y = 'sepal_width',data = iris)

可以看出这里小编使用了seaborn库中的relplot()函数，其实relplot()函数主要是用来绘制关系型图表的，其名字为relation+plot的复合。所以relplot函数被用来绘制散点图和线形图这两种关系型图表，默认情况下绘制散点图，当然也可以在函数参数中通过使用kind = scatter来进行说明，这种新的绘图方式是在seaborn0.8版本之后引入的，所以如果要使用这种语法方式请升级你的seaborn库，如果你不想升级的话，那么就用sns.scatter()函数，不过小编这里强烈推荐你使用relplot函数。从绘制的图形中可以看出，花萼的宽度与长度存在着一定程度的正相关。不过考虑到中文绘图的习惯，这里小编改一下坐标轴名称并同时增加一下图表名称。

sns.relplot(x = 'sepal_length',y = 'sepal_width',data=iris)
plt.title('seaborn与散点图初探')
plt.xlabel('花萼长度')
plt.ylabel('花萼宽度')

现在应该了解小编为什麼一定要介绍matplotlib库了吧，因为像seaborn这些高级库在绘图方面虽然存在着很大便利，但是在设置图形格式时还是会参照matplotlib的语法，因此小编也就比较习惯使用seaborn绘图，而用matplotlib设置格式（尽管seaborn自身也能设置格式，但是小编还是偏爱这种方式）。

散点图是由点对构成，散点图的根本就在于其中的每一个点，这就涉及到点的三种格式：颜色、大小和形状。所以散点图的美化也主要是从这三个方面展开，小编先来介绍颜色格式的设置。因为iris数据集中的species一列为各花的种类，所以小编这里就以这一分类特征为散点图着色。

sns.relplot(x = 'sepal_length',y = 'sepal_width',hue = 'species',data=iris)
plt.title('seaborn与散点图初探')
plt.xlabel('花萼长度')
plt.ylabel('花萼宽度')

在新的命令中仅仅是多了hue = 'species'这一参数，然而图形却发生了根本变化，这就是seaborn称之为高级绘图库的原因。可以看出图形按照花的类型分别进行了散点图的绘制，这一下可以更加明显地看出花萼宽度与花萼长度之间的正相关。seaborn的这种语法类似于R语言中的ggplot2语法，它也是将特征映射，而后根据映射进行绘图。其中的hue是色彩色调的意思，所以描述颜色的单词不只有colour，大家顺便复习一下自己的英语水平呀。下面接着讲述形状的设置。

sns.relplot(x = 'sepal_length',y = 'sepal_width',style = 'species',data=iris)
plt.title('seaborn与散点图初探')
plt.xlabel('花萼长度')
plt.ylabel('花萼宽度')

仅仅需要将参数中的hue换成style即可，当然你完全可以同时使用这两个参数，让图形既有颜色又有形状。

sns.relplot(x = 'sepal_length',y = 'sepal_width',hue = 'species',style = 'species',data=iris)
plt.title('seaborn与散点图初探')
plt.xlabel('花萼长度')
plt.ylabel('花萼宽度')

多加一个参数即可。这里需要特别说明一点的是，如果你同时使用颜色格式和形状格式，我建议你要让这两个参数的参数值最好相同，因为相对于形状，人眼对色彩更加敏感，所以如果你对这两个参数使用了不同的参数值，就会造成你观看图形时的视觉混乱，所以如果你不是火眼金睛最好还是老实一下吧。讲完了颜色和形状，那么接下来该谈一谈大小的问题了，在一般的绘图中小编其实不怎么使用大小这一格式的，因为散点图一旦使用了大小格式，那么它就是气泡图了，小编是不怎么喜欢使用气泡图的，不过这里还是介绍一下大小格式的设置。

sns.relplot(x = 'sepal_length',y = 'sepal_width',size = 'species',data=iris)
plt.title('seaborn与散点图初探')
plt.xlabel('花萼长度')
plt.ylabel('花萼宽度')

这里散点图大小的设置还是以species为基础的，所以同一类型花朵的点大小相同，这在美学角度是很不完美的，大小的设置就是为了使得点的大小能够反映值的大小，所以小编这里建议喜欢气泡图的朋友在设置大小这个格式的时候务必使用连续性变量。介绍完了散点图常用的三种格式设置，是时候进一步了。散点图一般用来反映变量之间的关系，所以很多时候散点图本身伴随有拟合线，这才是一副完美的散点图应该具有的特征，因此对于seaborn的介绍最后以这种图的绘制而告结束。

sns.lmplot(x = 'sepal_length',y = 'sepal_width',hue = 'species',data=iris)
plt.title('seaborn与散点图初探')
plt.xlabel('花萼长度')
plt.ylabel('花萼宽度')

这里命令函数换成了lmplot()，关于这一函数会在后续文章中作进一步说明，这里就不多讲了。通过这一函数，绘制了每种类别花朵的散点图分布与拟合情况，并同时使用不同的颜色加以了标注，很明显可以看出正相关关系是成立的。

介绍到这里基本上讲清楚了使用seaborn绘制散点图的基本流程，下面将绘图系统转向到R语言中的ggplot2包，小编将告诉你怎么使用ggplot2绘制这些散点图。

ggplot2与散点图

这一部分中，小编将主要以代码的形式给出，至于介绍环节则是能免则免。在正式开始之前小编强烈地建议你在R语言中安装tidyverse包，这个包将涵盖你学习数据分析时能够用到的全部R语言包，它就类似于python方面的Anaconda发行包。喝口凉水，让R语言代码直接开始吧！

# 检验R语言中是否安装了tidyverse包，如果没有安装则安装，而后加载这个包
if require(tidyverse){
    install.packages('tidyverse')
}
library(tidyverse)

# 绘制基本散点图，对应seaborn的图1
ggplot(data = iris) + geom_point(aes(x = iris$Sepal.Length,y = iris$Sepal.Width))

# 绘制seaborn的第二幅图
ggplot(data = iris) + geom_point(aes(x = iris$Sepal.Length,y = iris$Sepal.Width)) + labs(title = 'ggplot2与散点图',x = '花萼长度',y = '花萼宽度')

# 绘制第三幅图
ggplot(data = iris) + geom_point(aes(x = iris$Sepal.Length,y = iris$Sepal.Width,colour = iris$species)) + labs(title = 'ggplot2与散点图',x = '花萼长度',y = '花萼宽度')

# 绘制第四幅图
ggplot(data = iris) + geom_point(aes(x = iris$Sepal.Length,y = iris$Sepal.Width，shape = iris$species)) + labs(title = 'ggplot2与散点图',x = '花萼长度',y = '花萼宽度')

# 绘制第五幅图
ggplot(data = iris) + geom_point(aes(x = iris$Sepal.Length,y = iris$Sepal.Width，colour = iris$Species,shape = iris$Species)) + labs(title = 'ggplot2与散点图',x = '花萼长度',y = '花萼宽度')

#绘制第六幅图
ggplot(data = iris) + geom_point(aes(x = iris$Sepal.Length,y = iris$Sepal.Width，size = iris$species)) + labs(title = 'ggplot2与散点图',x = '花萼长度',y = '花萼宽度')

# 绘制第七幅图
ggplot(data = iris) + geom_point(aes(x = iris$Sepal.Length,y = iris$Sepal.Width)) + geom_smooth(aes(x = iris$Sepal.Length,y = iris$Sepal.Width,colour = iris$Species)) +labs(title = 'ggplot2与散点图',x = '花萼长度',y = '花萼宽度')

因为ggplot2相关的语法小编在这之前已有介绍，所以这里就不对这些命令多做说明了，大家可以参照着这些命令在R语言中尝试一下。
