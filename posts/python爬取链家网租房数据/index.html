python爬取链家网租房数据

August 20, 2018

写在前面

前期文章和大家聊了一下怎么爬取多页面网页，了解了爬虫的基本原理及步骤，那么本期小编准备再接再厉，介绍一下跨页面爬虫原理。考虑到八月将逝，九月即来，新的租房需求即将诞生，本期小编就来讲一下怎么使用python爬取链家网上面的租房信息，希望对大家的租房有所帮助。下面正式开始！

准备工作及需求分析

跨页面爬虫的基本原理步骤小编已经在前期文章中有所介绍，这里就不再多说。考虑到任意的爬虫都是提出网页需求，获得网页，而后解析网页获得相关数据，最后保存数据用于分析，跨页面爬虫自然也逃不出这个套路。只不过相对于多页面爬虫来说，跨页面爬虫需要多定义一个爬取网页链接的函数而已。本文仍然以python为工具详细介绍怎么使用python以及相关的库来爬取链家网的租房数据。在爬取租房数据开始之前，首先需要提出一个问题，在爬取过程中需要获得什麼信息？这里小编认为，既然要爬取租房的相关信息，房源名称自然是要需要的，房源的位置以及房价信息自然是少不了的，最终在考虑的诸多因素以及链家网的网页构造之后，小编决定获取十种租房信息：房源名称、房源所在小区、房源所在商圈、房源所在区域、房源价格、房源面积、户型、临近地铁、房源经纪以及相应的联系号码。考虑到小编住在广州且对广州情有独钟，这里的租房信息全部表现为广州区域的租房信息，当然你也可以通过替换小编的代码获取不同城市的租房信息。既然已经了解了需求，那么下面就要正式开始爬虫实战了。在爬取之前，让我们先导入相应的库。

import requests 
from bs4 import BeautifulSoup
import xlwings as xw
import time

前两个库是完成爬虫的主力，第三个库是用来将数据存入Excel之中，至于第四个库的导入主要是为了使用库中的休眠函数，以应对反爬。因为跨页面爬虫需要爬取网页链接以及爬取网页内的信息，所以下面需要定义两个函数，其中一个用来爬取网页链接，另一个用来爬取详细的网页数据。下面首先介绍第一个函数。

网页链接爬取函数构造

所谓网页链接，就是指当你点击之时会弹出另外的网页，链家网上面的租房信息就是这种构造。当你在主页中点击房源的名称之时，会弹出关于该房源的详细信息，你所需要的租房数据就保存在详细信息页面中，但是为了达到详细信息页面，你必须先获得达到这一页面的网页链接，这就是需要定义爬取网页链接函数的原因。当然，说的好像很高大上，但是不得不说爬取网页的链接是真的容易。

header = {'User-Agent': 'Mozilla/5.0 (Windows NT 6.1; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/xxxxxxxxx Safari/537.36'}   # 记得替换成你自己的请求头文件
def get_url(url):
    res = requests.get(url，header = header)
    bsobj = BeautifulSoup(res.text,'lxml')
    urls = bsobj.select('#house-lst > li > div.info-panel > h2 > a')

上面的函数可以爬取一页中所有房源的具体网页链接，可以看到上面并没有什麼新的命令，这些命令在前期的文章中都已有介绍，所以这里就不再赘述。通过这个函数，可以获得进入详细页面的链接，因此下一步就是需要构造一个爬取详细页面数据的函数。

爬取详细页面数据

def get_info(url):
    res = requests.get(url,header = header)
    bsobj = BeautifulSoup(res.text,'lxml')
    title = bsobj.find('h1',{'class':'main'}).get_text()
    price = bsobj.find('span',{'class':'total'}).get_text()
    area = bsobj.find_all('p',{'class':'lf'})[0].get_text()[3:]
    house_type = bsobj.find_all('p',{'class','lf'})[1].get_text()[5:]
    metor = bsobj.find_all('p')[4].get_text()[3:]
    park = bsobj.find_all('p')[5].get_text().split('\n')[0][3:]
    distrinct = bsobj.find_all('p')[6].get_text().split(' ')[0][3:]
    bankuai = bsobj.find_all('p')[6].get_text().split(' ')[1]
    broker = bsobj.find('div', {'class': 'brokerName'}).get_text().split('\n')[1] if bsobj.find('div', {'class': 'brokerName'}) is not None else 0
    phone = bsobj.find('div', {'class': 'phone'}).get_text().strip().replace('\n', '').replace(' ', '') if bsobj.find('div', {'class': 'phone'}) is not None else 0

在获取网页信息时，小编采用了find和find_all函数，这里有必要介绍一下这两个函数。这两个函数正如其名字所示，根据函数中的参数选择出相应内容，其中第一个参数为网页标签，第二个参数为属性对，一般来说在网络爬虫中熟练掌握这两个参数就可以了。这两个函数典型的不同点就在于，find函数选择出满足参数设定的第一条内容，而find_all函数则用来选择出所有满足参数值的内容，并将这些内容构成列表，所以使用find_all参数常常伴随有列表选择。至于参数值的设定，则需要你掌握一些基本的网页代码知识，这里小编就不多说了，感兴趣的朋友可以看一下相关的书籍。另外在代码中，小编也使用了一些字符串操作函数，主要有三个：split、replace以及strip。第一个用来根据指定符号将字符串分割中列表；第二个函数用来替换字符串中的内容；第三个函数用来消除字符串中的空格。这三个函数在网络爬虫中使用较为频繁，希望诸位朋友能够熟练掌握，关于这几个函数小编在前期的文章中也有所介绍，大家可以参看小编之前关于字符串介绍的那期文章。最后需要说明的是最后两行代码，这两行代码小编使用了二元选择表达式，因为某些房源并没有经纪人信息和联系电话方面的信息，所以为了避免这种情况的出现造成程序的中断，这里采用二元选择表达式构造了变量值，这种表达式在一些场景中极其好用。既然已经有了相应的网页链接爬取函数和具体网页信息获取函数，那么是时候将其整合在一起了，下面给出完整代码。

爬取链接租房数据的详细代码

# _*_ coding:utf-8 _*_
"""
    @Author: liupu
    @Filename: lianjia_zufang.py
    @Date: 2018-8-20
    @Description:
        # usage: 爬取链家网租房数据
        # platform: win7 + python3 + visual studio code
"""

# 导入关联库
import requests 
from bs4 import BeautifulSoup
import xlwings as xw 
import time

# 创建Excel文件，并命名标题行
wb = xw.Book()
sht = wb.sheets[0]
sht.range('A1').value = '房源名称'
sht.range('B1').value = '所在小区'
sht.range('C1').value = '所在商圈'
sht.range('D1').value = '区域'
sht.range('E1').value = '价格'
sht.range('F1').value = '面积'
sht.range('G1').value = '户型'
sht.range('H1').value = '地铁'
sht.range('I1').value = '经纪人'
sht.range('J1').value = '联系电话'

header = {'User-Agent': 'Mozilla/5.0 (Windows NT 6.1; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/xxxxxxxxx Safari/537.36'} 
# 构造爬取网页链接的函数
def get_html(url,pages):
    res = requests.get(url，header = header)
    bsobj = BeautifulSoup(res.text,'lxml')
    urls = bsobj.select('#house-lst > li > div.info-panel > h2 > a')
    location = 2
    for url in urls:
        url = url.get('href')
        get_info(url,location,page = pages)
        location += 1

# 构造爬取详细网页网页信息的函数
def get_info(url,location,page = 0):
    res = requests.get(url,header = header)
    bsobj = BeautifulSoup(res.text,'lxml')
    title = bsobj.find('h1',{'class':'main'}).get_text()
    price = bsobj.find('span',{'class':'total'}).get_text()
    area = bsobj.find_all('p',{'class':'lf'})[0].get_text()[3:]
    house_type = bsobj.find_all('p',{'class','lf'})[1].get_text()[5:]
    metor = bsobj.find_all('p')[4].get_text()[3:]
    park = bsobj.find_all('p')[5].get_text().split('\n')[0][3:]
    distrinct = bsobj.find_all('p')[6].get_text().split(' ')[0][3:]
    bankuai = bsobj.find_all('p')[6].get_text().split(' ')[1]
    broker = bsobj.find('div', {'class': 'brokerName'}).get_text().split('\n')[1] if bsobj.find('div', {'class': 'brokerName'}) is not None else 0
    phone = bsobj.find('div', {'class': 'phone'}).get_text().strip().replace('\n', '').replace(' ', '') if bsobj.find('div', {'class': 'phone'}) is not None else 0
# 存储数据到Excel中
    sht.range(page * 30 + location, 1).value = title
    sht.range(page * 30 + location, 2).value = park
    sht.range(page * 30 + location, 3).value = bankuai
    sht.range(page * 30 + location, 4).value = distrinct
    sht.range(page * 30 + location, 5).value = price
    sht.range(page * 30 + location, 6).value = area
    sht.range(page * 30 + location, 7).value = house_type
    sht.range(page * 30 +location, 8).value = metor
    sht.range(page * 30 +location, 9).value = broker
    sht.range(page * 30 +location, 10).value = phone

# 运行程序
if __name__ == '__main__':
    urls = ['https://gz.lianjia.com/zufang/pg{}'.format(ii) for ii in range(1,101)]
    for ii,url in enumerate(urls):
        get_html(url,ii)
        time.sleep(1) 
    wb.save('lianjia_zufang.xlsx')
    wb.close()

现在已经成功爬取了链家上面的租房信息，并成功地将其存放在了Excel文件中，后续便可以采用这些数据进行相应的分析了。另外程序的完整代码可以在作者的github获得，关于小编的github账号可以参看前期的文章。
