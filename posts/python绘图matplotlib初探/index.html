第一讲：matplotlib绘图准备

喜欢数据分析的朋友应该都知道在进行图表绘制时，最为方便的语言是R语言，R中自带的以及第三库绘图包极大地方便了各种绘图实践，这也使得“左手用R右手Python”成为数据分析领域的标志语言。相对于R语言中的绘图系统，python自身在绘图方面可以说就是一个小白，然而随着matplotlib库的开发以及基于matplotlib库而开发的其它绘图库，使得python的绘图能力得到了极大地提升，现在已经基本可以绘制你在工作中遇到的任意图表了，这也是笔者讲述python绘图系统会选择讲解matplotlib库的原因之一。

那么问题来了，为什麼这个库被命名为matplotlib呢，为什麼不直接命名plot，这该多方便呀？其实这源自于本库开发者的绘图经历，你如果稍稍地分解一些这个单词，就会看到matplotlib=mat+plot+lib，这是三个单词的合成词，第一个单词是另外一种著名编程语言matlab的缩写，第二个单词是绘图的英文，第三个单词是库的英文缩写，所以整合在一起，matplotlib便是基于matlab绘图语法开发出的python绘图库。这就是为什麼使用matplotlib进行绘图时会和使用matlab绘图很像的缘故，如果熟悉matlab的朋友对此应该不会太陌生，小编当初就是被matlab那种简洁的绘图语法所吸引才从Excel走向了程序之路，后来偏爱于R语言的数据统计分析与高效美观绘图从而转向了R语言，虽然现在工作实践以python为主，但是小编还是坚持每天会使用R语言，尤其是生成自动化报告这一块（这些笔者这里就不再多讲了），现在相信你已经了解了matplotlib库的来源。

因为matplotlib库来源于matlab绘图语法，所以matplotlib绘图时采用的各种绘图命令和matlab很相似；但是考虑到python自身的语法特征，matplotlib库又引入了第二套绘图语法，这套语法基于对象编程，可以说熟悉python的朋友会很喜欢这种方式。所以，现如今的matplotlib库存在着并行的两套绘图语法体系，笔者在这里不会刻意推荐你去学习哪一种，因为两种语法都有最适合使用之处，所以你完全可以交替或者互补地使用这两种语法，永远的一句真理之言就是能捉到老鼠的猫都是好猫，因此绘图语法的选择也是根据你使用哪一种更加方便进行选择。当然，小编前期会用两篇文章来依次介绍这两种绘图语法，而在之后的绘图中小编会根据场景选择语法体系。本期文章将先介绍matplotlib库的matlab式语法，后续的一篇文章会介绍面向对象的绘图语法体系。既然已经了解了matplotlib，那么接下来就开始相关的实践，在这之前确定你已经成功地在电脑上安装了matplotlib库。如果你安装的是anaconda软件包的话，就请忽略此处的安装问题，如果你安装的是原生态的python的话，那么你可以打开windows命令行（使用win+R进行打开）,然后在命令行中输入一下命令：

pip install matplotlib
# 或者可以使用下面命令（推荐使用第一种方式，第二种方式针对老版本的python）
easy_install matplotlib

这样就成功地安装了matplotlib，但是这种方式有时会出现一些异常情况从而导致安装不成功，所以笔者还是强烈地推荐你使用anaconda。好了，枪已磨好，路已建成，下面开战吧！

第二讲：matplotlib绘图实践

本期文章主要是对matplotlib库绘制图表有个俯瞰式的了解，因此本期文章将主要介绍怎么使用matplotlib库进行常用的一些商务图表的绘制，而对于图表的细节描述以及一些更加复杂的图形绘制会在以后的文章中逐步介绍。所以下面主要以日常工作最为常用的六种图表（折线图、散点图、饼图、直方图、条形图以及箱线图）来对matplotlib库的使用语法进行介绍。，对于这些图表，相信各位朋友都已比较熟悉，而且笔者在先前的一篇文章中也已有介绍，所以接下来在各图的绘制中就不再对其进行介绍，而主要是以代码加图形的方式给出一些内容，而对于绘图中所采用的数据会采用两种：一种是即时创建的数据；第二种是python库中的一些内生数据集，因此这里就不再给出相关数据了。同时笔者建议你在ipython或者jupyter notebook中进行以下的绘图实践，如果你在ipython命令行中执行，那么请打开windows命令行窗口，然后输入ipython --pylab；如果在jupyter notebook中执行，那么请在第一行中输入%matplotlib inline，这样的话你的图形就能实时显示，而不用在另外输入命令让其显示出来。在绘制各类图形之前，让我们先做一下准备，请在你的操作平台上输入以下代码：

import numpy as np
import matplotlib as mpl
import matplotlib.pyplot as plt
plt.rcParams['font.sans-serif']=['SimHei']
plt.rcParams['axes.unicode_minus']=False

第一行代码带人numpy库，以便生成绘图数据；第二行和第三行代码导入matplotlib库，其中第二行代码可以不输入，这只是笔者个人的习惯；最后两行代码用于解决绘图中的中文显示问题。下面进行折线图的绘制

折线图

x = np.linspace(0,np.pi,100)
y_cos = np.cos(x)

plt.plot(x,y_sin,label='$sin(x)$')
plt.title('三角函数')

前三行代码用来生成我们绘图用的数据，四五两行代码将两条这些绘制在一个图中；后面几行代码分别用来为图形设置标题、图例、轴坐标名。其中括在$$间的代码是LaTeX语法，如果你不了解LaTeX也没关系，尽情将其删去即可。图形显示如下： 上面绘制了折线图，考虑到面积图也属于折线图的一种，这里也在下面展示一下面积图的绘制方法。

第三讲：组合图绘制示例

上文对常用商务图表绘制方法进行了介绍，大致了解了这些商务图表的绘制方法，接下来小编更进一步将这些图表组合在一起，讲述组合图形的绘制方法，说到底仅仅是通过一条命令而已，各位可以在下面见到这一神秘命令。这里小编准备绘制四副图形的组合图，分别是折线图、散点图、直方图以及条形图（这是小编最为常用的集中图表，所以这里自然首选绘制它们了），废话不多说，直接上干货：

import datetime
start = datetime.datetime(2018,1,1)
end = datetime.datetime(2018,5,31)
delta = datetime.timedelta(days =1)
dates = mpl.dates.drange(start,end,delta)
x = np.random.rand(len(dates))
y1 = np.random.rand(5)*10
y2 = np.random.rand(5)*15
y3 = np.random.rand(5)*20
Index = np.arange(5)
rng = np.random.RandomState(0)
plt.subplot(2,2,1)
plt.plot_date(dates,x,'-')
plt.subplot(2,2,2)
for marker in ['o', '.', ',', 'x', '+', 'v', '^', '<', '>']:
    plt.plot(rng.rand(5), rng.rand(5), marker,label="marker='{0}'".format(marker))
plt.legend(numpoints=1)
plt.xlim(0,1.8)
plt.subplot(2,2,3)
plt.hist(np.random.normal(100,15,10000),bins=35,color='b')
plt.subplot(2,2,4)
plt.bar(Index-0.2,y1,width=0.2,color='r')
plt.bar(Index,y2,width=0.2,color='y')
plt.bar(Index+0.2,y3,width=0.2,color='b')
plt.suptitle('My favorate graph')

上面代码绘制了一个组合图，其中很多代码在前面已经讲述过，这里就不再赘述，需要说明的一条绘图命令是plt.subplot(row,col,i)，这条命令用来绘制组合图，前两个参数说明绘制几行几列图形，最后一个参数是指当前绘制第几个图形，小编上面共绘制了2行2列图形，所以使用命令为plt.subplot(2,2)。
